## Getting Started

### Clone the Repo

```bash
git clone https://github.com/santdeleon/react-starter.git

```

### Install Dependencies

```bash
yarn

```

### Run the App

```bash
yarn start

```

### Run Tests

```bash
yarn test
yarn test:ci

```
